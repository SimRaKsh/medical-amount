# medical-amount

AI-powered backend service to extract financial amounts from medical bills or receipts.

## Features
- OCR using EasyOCR
- Image + text input support
- OCR numeric error correction (O→0, I→1, l→1)
- Context-based classification (total, paid, due)
- Final structured JSON output with provenance
- Guardrails for noisy documents
